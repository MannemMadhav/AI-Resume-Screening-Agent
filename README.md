# AI Resume Screening Agent

An AI-powered ATS Resume Screening application built using Flask and Groq AI.

## Features

- Resume PDF Upload
- ATS Match Score
- Resume Category Detection
- ATS Recommendation
- Interview Questions
- Improvement Suggestions
- PDF Report Generation
- Analysis History Dashboard

## Technologies Used

- Python
- Flask
- Groq API
- PyPDF2
- ReportLab
- HTML
- CSS

## Run Locally

```bash
pip install -r requirements.txt
python app.py
